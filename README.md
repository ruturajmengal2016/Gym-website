## Run Locally

Clone the project

```bash
  git clone https://github.com/ruturajmengal2016/Gym-website.git
```

Go to the project directory

```bash
  cd Gym-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React, MUI, axios

**Server:** Node, Express
