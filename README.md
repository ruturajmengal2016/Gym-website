## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/ruturajmengal2016/Gym-website.git
```

Go to the project directory

```bash
  cd Gym-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
